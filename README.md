This repository contains Python scripts developed for Module 2: Basic Python Concepts. These programs demonstrate how to handle user input, perform mathematical calculations, and manipulate strings.

How to Run

Task 1: Basic Mathematical Operations
This script (basiccalsulations.py) takes two numbers from the user and performs fundamental arithmetic.

Functionality:
Accepts two numerical inputs.

Calculates the Addition, Subtraction, Multiplication, and Division.

Displays the results clearly on the screen.

Example Output:

Enter the first number: 5
Enter the second number: 10
Addition: 15
Subtraction: -5
Multiplication: 50
Division: 0.5

Task 2: Personalized Greeting
This program (greeting.py) focuses on string concatenation and user interaction.

Functionality:
Prompts the user for their first and last name.

Concatenates the first and last name into a full name.

Prints a personalized greeting message.

Example Output:

Enter your first name: John
Enter your last name: Doe
Hello, John Doe! Welcome to the Python program.

How to Run

Ensure you have Python installed on your system.

Clone this repository to your local machine.

Run the scripts using:

Bash
python basiccalsulations.py
python greeting.py
